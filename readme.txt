Adds a simple lightbox to default Gutenberg gallery.

== Description ==

In the default Gutenberg gallery, whenever you click a gallery image, you navigate to another page and see the linked image. This plugin show these images in a lightbox on the same page.

== Installation ==

1. Upload the plugins driectory and activate.
2. No extra steps or setup is needed.
3. You can easily style your lightbox in your own CSS file.

== Frequently Asked Questions ==

= Can you navigate to other gallery images, once you are in the lightbox? =

Not yet. We plan to add it in the near future.
